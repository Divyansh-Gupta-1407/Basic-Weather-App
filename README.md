# 🌤️ Weather Dashboard

A simple and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. The application fetches real-time weather information using the **OpenWeatherMap API** and displays key weather details in a clean, modern dashboard.

---

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Displays current temperature
* 💧 Shows humidity
* 🌬️ Displays wind speed
* ☁️ Shows current weather condition
* 📱 Responsive design for desktop and mobile
* 🎨 Modern UI with rounded cards, soft colors, and hover effects

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### 2. Open the project

Open the project folder in your preferred code editor.

### 3. Add your API Key

Open `script.js` and replace:

```javascript
appid = "YOUR_API_KEY"
```

with your own OpenWeatherMap API key.

### 4. Run the application

Open `index.html` in your browser.

---

## 📖 API Used

This project uses the **OpenWeatherMap Current Weather API**.

You can obtain a free API key by creating an account on the OpenWeatherMap website.
